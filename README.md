# Ruffle Web Pollyfill

This repository provides an template for running flash games with Ruffle. Ruffle is an open-source Flash Player emulator that allows you to continue running Flash content in modern browsers.

## Compatibility
Ruffle is designed to work in modern browsers that lack built-in Flash support, such as Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari. It is compatible with a wide range of Flash content, including animations, games, and interactive applications.

## Getting Started

To use Ruffle, follow these steps:

1. Use this as a template and create a new repository.
2. Delete "example.swf"
3. Upload the swf file of your choice.
4. Open index.html
5. Change the title "Ruffle" to the name of your choice.
```html
<title>Ruffle</title>
```
4. Replace "example.swf" to the name of the file you uploaded.
5. Save the changes to the index.html file.
6. Set up gitpages and you should now see your Flash content rendered using Ruffle.
7. You can now delete "README.md" to get rid of this
